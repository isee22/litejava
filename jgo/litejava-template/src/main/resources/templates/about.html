关于 LiteJava

LiteJava 是一个轻量级 Java Web 框架，灵感来自 Go 语言的 Gin 框架。

核心特性

  • Koa-style 洋葱中间件模型
  • Gin-style 路由（分组、通配符、参数）
  • 内置 JSON 处理（零依赖）
  • 插件系统（数据库、缓存、模板等）
  • 配置系统（.properties / YAML）
  • 核心模块零外部依赖

技术栈

  • JDK 1.8+
  • MyBatis
  • Thymeleaf
  • HikariCP
